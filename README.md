# Strona_003

Zmienne pod SEO i dane firmy można ustawić w Netlify jako Environment Variables.
Jeśli którejś brakuje, aplikacja użyje placeholderów z fallbacku.

Obsługiwane zmienne:

- `PUBLIC_SEO_TITLE`
- `PUBLIC_SEO_DESCRIPTION`
- `PUBLIC_SITE_NAME`
- `PUBLIC_SITE_URL`
- `PUBLIC_COMPANY_NAME`
- `PUBLIC_COMPANY_TAGLINE`
- `PUBLIC_COMPANY_PHONE`
- `PUBLIC_COMPANY_EMAIL`
- `PUBLIC_COMPANY_STREET`
- `PUBLIC_COMPANY_CITY`
- `PUBLIC_COMPANY_ZIP`
- `PUBLIC_COMPANY_FACEBOOK`
- `PUBLIC_COMPANY_INSTAGRAM`
- `PUBLIC_COMPANY_LINKEDIN`
